////////////////
// HOW TO RUN //
////////////////

1. EXTRACT the project folder 'boilerplate' 
2. COMPILE program using the command line 'make'
3. RUN the program using the command line './boilerplate.out

////////////////
// HOW TO USE //
////////////////

[R] reset to default settings
[C] toggle direction of smaller circle rotation
[P] pause/resume animation
[A] toggle animation on or off
[,] rotate entire hypocyclide counterclockwise
[.] rotate entire hypocyclide clockwise
[LEFT/RIGHT ARROW] adjusts number of cycles (n)
[UP/DOWN ARROW] adjusts speed of animation
<Scroll Wheel> adjust the zoom of the hypocycloid
//MAKE SURE NUMLOCK IS OFF
[NUM 7/4] adjusts size of small circle
[NUM 1] reset size of small circle
[NUM 5/8] adjusts size of big circle
[NUM 2] reset size of big circle
[ESCAPE] closes the program
